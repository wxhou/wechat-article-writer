# 公众号文章创作 Skill

自动化公众号文章创作与发布。

## 核心原则

**减少 AI 味**：
- 避免套路句式（首先/其次/最后）
- 增加具体案例和细节
- 口语化表达
- 真实观点

## 功能

- 自动场景识别（X链接/英文/标题）
- 质量检查 + AI 味检测
- AI配图 + 封面图
- 一键发布

## 使用

```bash
/wechat-article-writer <X链接/标题/英文文章>
```

## 依赖

- baoyu-post-to-wechat
- baoyu-cover-image
- baoyu-article-illustrator
