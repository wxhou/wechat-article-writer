# 公众号文章创作 Skill

自动化公众号文章创作与发布。

## 功能

- 自动场景识别（X链接/英文/标题）
- 自动化质量检查（错别字/可读性/敏感词）
- AI配图 + 封面图生成
- 一键发布到公众号

## 流程

```
自动识别 → 获取内容 → 质量检查 → 写作 → 配图+封面图 → 格式优化 → 发布
```

## 使用

```bash
/wechat-article-writer <X链接/标题/英文文章>
```

## 依赖

- baoyu-post-to-wechat
- baoyu-cover-image
- baoyu-article-illustrator
