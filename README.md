# 公众号文章创作 Skill

自动化公众号文章创作与发布。

## 核心原则

**你的写作风格**：
- 开头：反问/故事/个人经历
- 中间：口语化 + 个人踩坑经历
- 结尾：互动感
- 避免 AI 味句式

## 功能

- 自动场景识别
- AI 味检测（禁止首先/其次/最后）
- 个人经历融入
- AI配图 + 封面图
- 一键发布

## 使用

```bash
/wechat-article-writer <X链接/标题/英文文章>
```

## 依赖

- baoyu-post-to-wechat
- baoyu-cover-image
- baoyu-article-illustrator
